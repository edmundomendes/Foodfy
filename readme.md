# :gear: Como usar
```bash
# Clone este repositório
$ git clone https://github.com/edmundomendes/Foodfy.git

# Va até o repositório
$ cd Foodfy

# Instale as dependências do projeto
$ npm install
$ npm nodemon
$ npm run-all browser-sync

# Rode o servidor
$ npm start

# Acesse na web
http://localhost:3000

```
